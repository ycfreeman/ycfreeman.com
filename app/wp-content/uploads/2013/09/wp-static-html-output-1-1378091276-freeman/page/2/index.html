WOW Server Status WordPress Widget 1.0.7

June 8, 2011php, server status, widget, wordpress plugin, world of warcraft, wow realm status, wow-server-status-widgetJSON, php, plugin, realm status, server status, widget, wordpress, world of warcraftfreeman

Thanks Aoife for the info, I didn’t know Blizzard released their official API.

This plugin is now updated to use official JSON feed, (finally :p), php 5.2+ required (for native json_decode() support), although fallback JSON.php for earlier versions is included as well, no garrantee it works in 5.2-, too bad if it doesn’t work .

For non-wordpress users, feel free to download the package and extract the wow_ss.php to use it like the original one. (text display mode may contain bugs), more information about this plugin
